# idsec_cce_aws_workspaces (Data Source)

CCE AWS Workspaces data source, retrieves AWS organizations and accounts with filtering.

## Example Usage

```terraform
data "idsec_cce_aws_workspaces" "all_workspaces" {
  # Optional filters:
  # include_suspended = true
  # workspace_type = "aws_account"
  # workspace_status = "Completely added"
  # parent_id = "aaaa1111bbbb2222cccc3333dddd4444"
  # services = "dpa,sca"
}
```
