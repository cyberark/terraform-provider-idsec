# idsec_identity_policies_order (Data Source)

The Identity service policies order data source. It reads the order of policies and is based on the ID of the policy order configuration.

## Example Usage

```terraform
data "idsec_identity_policies_order" "my_policy_order" {}


data "idsec_identity_policies_order" "my_policy_order_specific" {
  policies_order = [
    "policy_id_1",
    "policy_id_2",
    "policy_id_3"
  ]
}
```
