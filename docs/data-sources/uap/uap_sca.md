# idsec_uap_sca (Data Source)

Cloud console access policy data source - enables creating, viewing,editing, and deleting access policies to cloud services. <br>For more information about the schema parameters, see the <a href='https://api-docs.cyberark.com/uap-schema-api/docs/access-control-policies-api' target='_blank'>Access Control Policies API</a> documentation.

## Example Usage

```terraform
data "idsec_uap_sca" "example_policy" {
  policy_id = "example_policy_id"
}
```
