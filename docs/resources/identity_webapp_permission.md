# idsec_identity_webapp_permission (Resource)

The Identity service webapp permission resource that is used to manage webapp permissions.

## Example Usage

```terraform
resource "idsec_identity_webapp_permission" "my_webapp_permission" {
  webapp_id      = "my_app_id"
  principal      = "user@cyberark.cloud.12345"
  principal_type = "User"
  rights = [
    "Admin",
    "Grant",
    "View",
    "ViewDetail",
    "Execute",
    "Automatic",
    "Delete"
  ]
}
```
