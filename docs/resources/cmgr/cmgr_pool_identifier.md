# idsec_cmgr_pool_identifier (Resource)

Connector manager pool identifier resource, is associated to a pool for identifying the pool in a simplified manner, and not only via the network name

## Example Usage

```terraform
resource "idsec_cmgr_pool_identifier" "example_identifier" {
  type    = "GENERAL_FQDN"
  value   = var.pool_identifier_fqdn
  pool_id = var.pool_id
}
```
