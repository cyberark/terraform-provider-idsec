# idsec_pcloud_application (Resource)

pCloud application resource, manages pCloud applications information / metadata and credentials.

## Example Usage

```terraform
resource "idsec_pcloud_application" "myapp" {
  app_id                = "MyApp"
  business_owner_f_name = "User"
  business_owner_l_name = "Name"
  business_owner_email  = "user.name@example.com"
}
```
